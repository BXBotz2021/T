# Telegraph Uploader Bot
A telegram media to telegraph uploader bot

---

## Variables

- `API_HASH` Your API Hash from my.telegram.org
- `API_ID` Your API ID from my.telegram.org
- `BOT_TOKEN` Your bot token from @BotFather
